## Run Locally

Clone the project

```bash
  git clone https://github.com/1Mohitpl/Nodejs_project.git
```

Go to the project directory

```bash
   cd Project1
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
